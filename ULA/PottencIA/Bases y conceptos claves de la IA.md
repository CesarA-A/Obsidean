## Tipos de algoritmos de los que aprenden las IA

- **Sistemas basados en conocimiento:** Almacena información de un tema en particular para resolver problemas.
- **Aprendizaje automático**: Aprenden por si mismos, identificando patrones.
- **Aprendizaje Profundo**: Usan redes neuronales con múltiples capas para poder hacer cosas mas complejas, es una forma avanzada de ML
- **Aprendizaje bioinspirados**: Se basan en sistemas biológicos. (Algoritmos genéticos, se usan para optimar robots que imitan animales o en programación de enjambre)

**Unidades de procesamiento gráfico (GPU)**:Son esenciales para la IA por que pueden hacer múltiples operaciones simples a la vez.Esto las hace ideales en el entrenamiento de los modelos.

**Estadística y probabilidad**:La estadística para encontrar los patrones, analizar y construir modelos, la probabilidad nos ayuda a modelar eventos posibles esto para que las IA tome decisiones.

## Tipos de IA

- **IA débil (estrecha)**:Se usa para tareas especificas.
- **IA fuerte (AGI,Generativa Artificial)**:Es un concepto hipotético que dice que existe una IA con capacidades de un cerebro humano.
- **Super IA**:IA que supera a las capacidades de un cerebro humano.

##  Categorías

- **Aprendizaje Supervisado**: El algoritmos se entrena  con datos etiquetados (nombrados) de entradas y salida esto por que después pueda dar la salida.
- **Aprendizaje No Supervisado**: Se entrena con datos no etiquetados esto para que pueda encontrar patrones.
		Modelos
	1. Clustering: Organiza datos similares en grupos.
	2. Reducción de Dimensiones: Encontrar una representación mas pequeña de los datos.
- **Aprendizaje por Refuerzo**: Se entrena al algoritmo mediante sus acciones si cumple con lo que deseamos le damos una recompensa de lo contrario se le castigo. Se usa en robotica y videojuegos.



**Pasos clave para cualquier aprendizaje**

a) Recopilación y procesamiento: Es procesamiento se utiliza para limpiar los datos, manejas valores faltantes, normalizarlos y darles formatos.

>[!Bug] Datos
>Los datos tiene que estar libres de sesgos esto por que los modelos podrían tener esos mismo sesgos.


b)Selección y extracción de características:
	Selección: Elegir los atributos que no importan
	 Extracción: Crear nuevas características a partir de las que conocemos.

c) Elegir algoritmos: Se suele hacer pruebas con diferentes tipos da algoritmos para escoger el que convenga.

d) Evaluar el modelo:
	Se  evalúa con la siguiente métricas:
	- Presión
	- Recall.
	- Error cuadrático medio.

e) Despliegue del algoritmo:Se debe monitoriar el modelo y actualizarlo de manera periodica.
